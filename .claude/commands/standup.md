# /standup — Daily Status

Give a quick status update on where this project stands right now.

1. Run `git log --oneline -5` to see recent commits
2. Run `git status` to see current state
3. Run `npm test 2>&1 | tail -5` to check test status
4. Check `docs/HANDOFF.md` if it exists

Then report:

**Yesterday / last session:**
- [What was completed]

**Today / this session:**
- [What's the plan]

**Blockers:**
- [Anything blocking progress]

**Tests:** [passing / X failing]
**Branch:** [current branch]

Keep it short — this is a standup, not a status report.
